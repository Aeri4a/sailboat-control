# Sailboat control

## Web app
### Initializing
```
cd app
npm i

cd simulation
npm i
```
### Starting App
```
cd app
npm start
```

### Starting backend for simulation data
```
cd simulation
npm start
```

## Technologies
- Python
- React (TypeScript)
- NodeJS (TypeScript)
